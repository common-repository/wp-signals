== Installation ==
 
Just install this plugin as you would do any other - either through Wordpress plugin directory, or by uploading the zip
in your WP admin panel.

This plugin is free to use, and contains a number of awesome features. Those include integrations with other
WordPress plugins, setting up server-side events, and powerful analytics tools.


== Frequently Asked Questions ==
 
= What is a Facebook Pixel? =
A Facebook pixel helps you measure customer actions, build audiences and unlock optimization tools for your ads.

= How can I add my first pixel? =
The simplest way to add a FB pixel with the WP Signals plugin is to use our powerful wizard, which you can access on the "Settings" tab.

= What is the Conversions API? =
The Conversions API allows advertisers to send web events from their servers directly to Facebook, skipping any roadblocks set by browsers and ad blockers. WP Signals supports all features from the Conversions API.
